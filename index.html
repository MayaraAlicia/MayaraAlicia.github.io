MEUS PROJETOS

Jogo da Forca

Implementação do Jogo da Forca com foco na lógica de programação. O script utiliza a biblioteca random para selecionar palavras de listas pré-definidas e gerencia o estado do jogo, controlando as vidas, as letras já tentadas e a exibição da palavra oculta.

Consultor de clima

Um script de console que usa a API do OpenWeatherMap para buscar o clima atual de uma cidade.

Jogo Atlas

Jogo de terminal em Python que testa o conhecimento geográfico do usuário.
O script consome a API REST (RestCountries) para buscar, filtrar e validar uma lista de países independentes.
Usei a biblioteca requests para a conexão e sets para gerenciar as respostas e a pontuação final.

Jogo 21 (Blackjack)

Desenvolvimento de um jogo de Blackjack em Python, onde o jogador compete contra o computador. O jogo inclui funcionalidades como distribuição de cartas, cálculo de pontuação e lógica para determinar o vencedor, proporcionando uma experiência interativa e divertida.
